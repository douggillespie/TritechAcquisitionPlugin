Tritech Sonars

Overview

The Tritech Sonars modules can be used to acquire and process data from Tritech Multibeam imaging sonars.

Currently, three modules are available to

  • Acquire sonar data (in real time or from file, just like the PAMGuard acquisition module)
  • Automatically detect tracks in the sonar data
  • Record sonar data to archive files for later analysis

Installation

All three modules are packaged into a single PAMguard plugin, called something like TritechAcquisitionV*_**.jar, where *_** is the version number. This does not come as a standard part of a PAMGuard installation. Read / copy the stuff in the released zip file. To add Tritech modules to an installation, you should add the latest TritechAcquisitionV*_**.jar file to the plugins folder of your PAMGuard installation. The modules will then appear the next time you start PAMGuard.
