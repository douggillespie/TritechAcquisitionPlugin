Tritech Sonars

Overview

The Tritech Sonars modules can be used to acquire and process data from Tritech Multibeam imaging sonars.

Currently, three modules are available to

  • Acquire sonar data (in real time or from file)
  • Automatically detect tracks in the sonar data
  • Record sonar data to archive files for later analysis

Installation

If you're reading this in the PAMGuard Help, it's likely that you've already installed everything and don't need to do any more. If you're updating, or are having difficulties, then read on.

All three modules are packaged into a single PAMguard plugin, called something like TritechAcquisitionV*_**.jar, where *_** is the version number. This does not come as a standard part of a PAMGuard installation.

To acquire data directly from sonars, you also need some Windows library files to manage communication between PAMGuard and the Tritech devices.

Both the TritechAcquisitionV*_**.jar plugin and the required libraries are distributed together in zip archives through GitHub

Follow the instructions for the latest GitHub release ensuring that you copy files to the correct folders in your PAMGuard installation.
